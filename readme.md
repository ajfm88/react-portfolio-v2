## Live Demo:

[Live Version](https://www.ajfm88.com)

## Features:

- Uses ReactJS. ⚛️
- Styled with TailwindCSS. 🎨

## Installation:

- Clone the repo.
- Use the command `yarn` to install dependencies.
- Use the command `npm run start` to run the app in localhost.
